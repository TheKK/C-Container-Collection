(dllst)		 constructorvoidcc_dllst (Object, TYPE); Create an empty dllst container.
destructorvoidcc_dllst_dealloc (Object); Deallocate the container manually.
iteratorsvoidcc_dllst_iter (Iter, Object); Create an iterator for the dllst container.
boolcc_dllst_iter_is_valid (Iter); Return TRUE iff the iterator is dereferenceable.
NOTE: This does NOT guarantee the existence of the element.
TYPE&cc_dllst_iter_dereference (Iter); Return the reference to the element if the iterator is dereferenceable.
Return NULL otherwise.
voidcc_dllst_iter_copy (Iter_Dst, Iter_Src); Copy one iterator to the other.
Ignore the return value.
voidcc_dllst_iter_clear (Iter); Make the iterator not dereferenceable.
Ignore the return value.
Itercc_dllst_iter_begin (Iter, Object); Return and set the iterator to point to the first element.
Return NULL if the container is empty.
Itercc_dllst_iter_end (Iter, Object); Return and set the iterator to point to the last element.
Return NULL if the container is empty.
Itercc_dllst_iter_incr (Iter); Return and move forward the iterator by one element.
Return NULL if the iterator doesn't point to any element.
Itercc_dllst_iter_decr (Iter); Return and move backward the iterator by one element.
Return NULL if the iterator doesn't point to any element.
traversalSTATcc_dllst_trav (Object, Iter) { ... } Traversal all elements from the beginning to the end.
NOTE: This is NOT a function but a for-loop statement.
STATcc_dllst_trav_rev (Object, Iter) { ... } Traversal all elements from the end to the beginning.
NOTE: This is NOT a function but a for-loop statement.
capacityintcc_dllst_size (Object); Return the number of the elements in the container.
boolcc_dllst_empty (Object); Return TRUE iff the container is empty.
modifiersvoidcc_dllst_push_front (Object, Value); Insert an element at the beginning.
voidcc_dllst_push_back (Object, Value); Insert an element at the end.
