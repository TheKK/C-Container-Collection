(dllst)		 ConstructorDEFINEcc_dllst (TYPE) DLLST; Create a dllst container.
Use a C struct to simulate a container.
IteratorsDEFINEcc_dllst_iter (ITER, DLLST); Create an iterator for the dllst container.
The iterator might be invalid if the container is modified.
TYPE&cc_dllst_iter_dref (ITER); Return a reference to the element.
It's an undefined behavior if the iterator is not invalid.
voidcc_dllst_iter_copy (ITER, ITER); Assign the right iterator to the left iterator.
It's not acceptable to assign the iterator by assignment operator.
voidcc_dllst_iter_head (ITER, DLLST); Set the iterator to the head.
The head is a sentinel node that points to the first element.
voidcc_dllst_iter_tail (ITER, DLLST); Set the iterator to the tail.
The tail is a sentinel node that points to the last element.
voidcc_dllst_iter_begin (ITER, DLLST); Set the iterator to the first element usaully.
Set the iterator to the tail if the container is empty.
voidcc_dllst_iter_end (ITER, DLLST); Set the iterator to the last element usaully.
Set the iterator to the head if the container is empty.
void*cc_dllst_iter_incr (ITER); Move forward the iterator by one element.
Return NULL iff the iterator doesn't point to any element.
void*cc_dllst_iter_decr (ITER); Move backward the iterator by one element.
Return NULL iff the iterator doesn't point to any element.
TraversalLOOPcc_dllst_trav (DLLST, ITER) { ... } Traversal all elements from the beginning to the end.
There is no effect if the container is empty.
LOOPcc_dllst_trav_backward (DLLST, ITER) { ... } Traversal all elements from the end to the beginning.
There is no effect if the container is empty.
AccessTYPE&cc_dllst_front (DLLST); Return a reference to the first element.
It's an undefined behavior if the container is empty.
TYPE&cc_dllst_back (DLLST); Return a reference to the last element.
It's an undefined behavior if the container is empty.
Capacityintcc_dllst_size (DLLST); Return the number of the elements in the container.
Return 0 if the container is empty.
boolcc_dllst_empty (DLLST); Return TRUE iff the container is empty.
TRUE is usually represented by any numeric value not equal to zero.
ModifiersSTATcc_dllst_push_front (DLLST, value); Insert an element at the beginning.
This makes a copy but not a reference of the value into the container.
STATcc_dllst_push_back (DLLST, value); Insert an element at the end.
This makes a copy but not a reference of the value into the container.
STATcc_dllst_pop_front (DLLST); Remove the first element.
There is nothing modified if the container is empty.
STATcc_dllst_pop_back (DLLST); Remove the last element.
There is nothing modified if the container is empty.
STATcc_dllst_clear (DLLST); Remove all elements in the container.
This does not deallocate all elements in the container.
DestructorSTATcc_dllst_free (DLLST); Deallocate all elements in the container manually.
The constructor and destructor should exist in pairs.
MISC. Constants#defineCCC_DLLST_BLOCK_SIZE NUMBER Set the number of elements that a single block in the memory pool can hold.
The default value is 16.
