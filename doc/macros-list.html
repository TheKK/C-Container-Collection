(dllst)		 constructor<DECL>cc_dllst (Object, TYPE); Create a dllst container.
Use aligned structure to enhance performance.
<DECL>cc_dllst_packed (Object, TYPE); Create a packed dllst container.
Use packed structure to reduce memory usage.
destructor<STAT>cc_dllst_free (Object); Deallocate the container manually.
The constructor and destructor should exist in pairs.
iterators<DECL>cc_dllst_iter (Iter, Object); Create an iterator for the (packed) dllst container.
The iterator might be invalid if the container is modified.
TYPE&cc_dllst_iter_deref (Iter); Return a reference to the element.
It's an undefined behavior if the iterator is not dereferenceable.
voidcc_dllst_iter_copy (Iter, Iter); Assign the right iterator to the left iterator.
It's not acceptable to assign the iterator by equality sign.
voidcc_dllst_iter_head (Iter, Object); Set the iterator to the head.
Note: The head is the previous node of the first element.
voidcc_dllst_iter_tail (Iter, Object); Set the iterator to the tail.
Note: The tail is the next node of the last element.
voidcc_dllst_iter_begin (Iter, Object); Set the iterator to the first element.
Set the iterator to the tail if the container is empty.
voidcc_dllst_iter_end (Iter, Object); Set the iterator to the last element.
Set the iterator to the head if the container is empty.
<EXPR>cc_dllst_iter_incr (Iter); Move forward the iterator by one element.
Evaluate to NULL iff the iterator doesn't point to any element.
<EXPR>cc_dllst_iter_decr (Iter); Move backward the iterator by one element.
Evaluate to NULL iff the iterator doesn't point to any element.
capacityintcc_dllst_size (Object); Return the number of the elements in the container.
boolcc_dllst_empty (Object); Return TRUE iff the container is empty.
element
access		TYPE&cc_dllst_front (Object); Return a reference to the first element.
It's an undefined behavior if the container is empty.
TYPE&cc_dllst_back (Object); Return a reference to the last element.
It's an undefined behavior if the container is empty.
modifiers<STAT>cc_dllst_push_front (Object, Value); Insert an element at the beginning.
<STAT>cc_dllst_push_back (Object, Value); Insert an element at the end.
<STAT>cc_dllst_pop_front (Object); Remove the first element if the container is not empty.
<STAT>cc_dllst_pop_back (Object); Remove the last element if the container is not empty.
<STAT>cc_dllst_clear (Object); Remove all elements in the container without deallocating.
traversal<LOOP>cc_dllst_trav (Object, Iter) { ... } Traversal all elements from the beginning to the end.
<LOOP>cc_dllst_trav_rev (Object, Iter) { ... } Traversal all elements from the end to the beginning.
settings flags#defineCCC_DLLST_BLOCK_SIZE <NUMBER> Set the number of nodes that a single block in the memory pool can hold.
