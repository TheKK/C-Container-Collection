Container Operation Short Description
CategoryFunction-like Macros Prototype
DLLST Create defdllst (TYPE) DLLST; Create a dllst container of type TYPE.
Use a C struct to simulate a list container.
defdllst_iter (TYPE) ITER; Create an iterator for the dllst container of type TYPE.
The iterator might be invalid if the container is modified.
defdllst_pckd (TYPE) DLLST; Create a packed dllst container of type TYPE.
This decreases memory usage as much as possible.
defdllst_iter_pckd (TYPE) ITER; Create an iterator for packed dllst containers of type TYPE.
This should be used by every packed dllst container to make sure the correctness.
Initialize voiddllst_init (DLLST); Initialize the dllst container.
CAUTION: Every container should be initialized right after its creation.
voiddllst_iter_init (ITER, DLLST); Initialize the iterator for the dllst container.
CAUTION: Every iterator should be initialized right after its creation.
Destroy statdllst_free (DLLST); Deallocate all elements in the container manually.
CAUTION: Every container should be destroyed before the program terminates.
Access TYPE&dllst_front (DLLST); Return a reference to the first element.
It's an undefined behavior if the container is empty.
TYPE&dllst_back (DLLST); Return a reference to the last element.
It's an undefined behavior if the container is empty.
Capacity intdllst_size (DLLST); Return the number of the elements in the container.
Return 0 if the container is empty.
booldllst_empty (DLLST); Return TRUE iff the container is empty.
TRUE is usually represented by any numeric value not equal to zero.
Modifiers statdllst_push_front (DLLST, value); Insert an element at the beginning.
This makes a copy of value into the container.
statdllst_push_back (DLLST, value); Insert an element at the end.
This makes a copy of value into the container.
statdllst_pop_front (DLLST); Remove the first element.
There is nothing modified if the container is empty.
statdllst_pop_back (DLLST); Remove the last element.
There is nothing modified if the container is empty.
statdllst_clear (DLLST); Remove all elements in the container.
This does not deallocate all elements in the container.
statdllst_insert (ITER, value); Insert an element at the position where the iterator points.
This makes a copy of value into the container.
statdllst_erase (ITER); Erase an element at the position where the iterator points.
There is nothing modified if the container is empty.
Operations statdllst_move_range (ITER_P, ITER_L, ITER_R); Move the elements in the range [ITER_L, ITER_R) to position where ITER_P points.
These three iterators should be affiliated to the same dllst container.
statdllst_merge_range (ITER_L, ITER_M, ITER_R, ITER_X); Merge two sorted segments in the range [ITER_L, ITER_M) and [ITER_M, ITER_R).
This requires an auxiliary iterator ITER_X which is not in use currently.
statdllst_sort (DLLST, &(ITER_X[4])); Sort all elements with default comparator DLLST_DEFAULT_LEQ_COMPARATOR.
This requires a pointer pointing to an array of four auxiliary iterators ITER_X[4].
stat
dllst_sort_openmp (DLLST, &(DLLST_X[N]),
&(ITER_X[N][4]), N (= threads));		 Sort all elements with default comparator DLLST_DEFAULT_LEQ_COMPARATOR.
N is the max number of threads that will be used for OpenMP parallel computing.
Iterators TYPE&dllst_iter_dref (ITER); Return a reference to the element.
It's an undefined behavior if the iterator is not invalid.
voiddllst_iter_copy (ITER_DST, ITER_SRC); Copy the iterator from ITER_SRC to ITER_DST.
It's not acceptable to assign the iterator by assignment operator.
voiddllst_iter_head (ITER, DLLST); Set the iterator to the head.
The head is a sentinel node that points to the first element.
voiddllst_iter_tail (ITER, DLLST); Set the iterator to the tail.
The tail is a sentinel node that points to the last element.
voiddllst_iter_begin (ITER, DLLST); Set the iterator to the first element usaully.
Set the iterator to the tail if the container is empty.
voiddllst_iter_end (ITER, DLLST); Set the iterator to the last element usaully.
Set the iterator to the head if the container is empty.
void*dllst_iter_incr (ITER); Move forward the iterator by one element.
Return NULL iff the iterator doesn't point to any element.
void*dllst_iter_decr (ITER); Move backward the iterator by one element.
Return NULL iff the iterator doesn't point to any element.
statdllst_iter_advance (ITER, diff); Move the iterator by diff element(s). (set forward as positive)
The iterator will stop at the sentinel node if there is no element left to iterate over.
Traversal loopDLLST_TRAVERSAL (DLLST, ITER) stat; Traverse all elements from the beginning to the end.
There is no effect if the container is empty.
loopDLLST_BACKWARD_TRAVERSAL (DLLST, ITER) stat; Traverse all elements (backward) from the end to the beginning.
There is no effect if the container is empty.
