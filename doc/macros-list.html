Container Operation Short Description
CategoryFunction-like Macros Prototype
DLLST Create defcc_dllst (TYPE) DLLST; Create a dllst container.
Use a C struct to simulate a container.
defcc_dllst_iter (TYPE) ITER; Create an iterator for the dllst container.
The iterator might be invalid if the container is modified.
Initialize voidcc_dllst_init (DLLST); Initialize the dllst container.
CAUTION: Every container should be initialized right after its creation.
voidcc_dllst_iter_init (ITER, DLLST); Initialize the iterator for the dllst container.
CAUTION: Every iterator should be initialized right after its creation.
Destroy statcc_dllst_free (DLLST); Deallocate all elements in the container manually.
CAUTION: Every container should be destroyed before the program terminates.
Access TYPE&cc_dllst_front (DLLST) Return a reference to the first element.
It's an undefined behavior if the container is empty.
TYPE&cc_dllst_back (DLLST) Return a reference to the last element.
It's an undefined behavior if the container is empty.
Capacity intcc_dllst_size (DLLST) Return the number of the elements in the container.
Return 0 if the container is empty.
boolcc_dllst_empty (DLLST) Return TRUE iff the container is empty.
TRUE is usually represented by any numeric value not equal to zero.
Modifiers statcc_dllst_push_front (DLLST, value); Insert an element at the beginning.
This makes a copy of value into the container.
statcc_dllst_push_back (DLLST, value); Insert an element at the end.
This makes a copy of value into the container.
statcc_dllst_pop_front (DLLST); Remove the first element.
There is nothing modified if the container is empty.
statcc_dllst_pop_back (DLLST); Remove the last element.
There is nothing modified if the container is empty.
statcc_dllst_clear (DLLST); Remove all elements in the container.
This does not deallocate all elements in the container.
statcc_dllst_insert (ITER, value); Insert an element at the position where the iterator points.
This makes a copy of value into the container.
statcc_dllst_erase (ITER); Erase an element at the position where the iterator points.
There is nothing modified if the container is empty.
Operations statcc_dllst_move_range (ITER_P, ITER_F, ITER_L); Move elements in the range [ITER_F, ITER_L) to position where ITER_P points.
These three iterators should be affiliated to the same dllst container.
Iterators TYPE&cc_dllst_iter_dref (ITER) Return a reference to the element.
It's an undefined behavior if the iterator is not invalid.
voidcc_dllst_iter_copy (ITER_DST, ITER_SRC) Copy the iterator from ITER_SRC to ITER_DST.
It's not acceptable to assign the iterator by assignment operator.
voidcc_dllst_iter_head (ITER, DLLST) Set the iterator to the head.
The head is a sentinel node that points to the first element.
voidcc_dllst_iter_tail (ITER, DLLST) Set the iterator to the tail.
The tail is a sentinel node that points to the last element.
voidcc_dllst_iter_begin (ITER, DLLST) Set the iterator to the first element usaully.
Set the iterator to the tail if the container is empty.
voidcc_dllst_iter_end (ITER, DLLST) Set the iterator to the last element usaully.
Set the iterator to the head if the container is empty.
void*cc_dllst_iter_incr (ITER) Move forward the iterator by one element.
Return NULL iff the iterator doesn't point to any element.
void*cc_dllst_iter_decr (ITER) Move backward the iterator by one element.
Return NULL iff the iterator doesn't point to any element.
statcc_dllst_iter_advance (ITER, diff); Move the iterator by diff element(s). (set forward as positive)
The iterator will stop at the sentinel node if there is no element left to iterate over.
Traversal loopcc_dllst_trav (DLLST, ITER) { ... } Traverse all elements from the beginning to the end.
There is no effect if the container is empty.
loopcc_dllst_trav_back (DLLST, ITER) { ... } Traverse all elements (backward) from the end to the beginning.
There is no effect if the container is empty.
Preset

Constants		#defineCC_DLLST_START integer Set integer as the number of reserved elements in the initial block (in stack).
The default value is 16.
#defineCC_DLLST_RATIO integer Set integer as the increment ratio for the next block if the current one is full.
The default value is 2.
#defineCC_DLLST_THRSH integer Set integer as the threshold value that the size of block will stop increasing at.
The default value is 65536.
